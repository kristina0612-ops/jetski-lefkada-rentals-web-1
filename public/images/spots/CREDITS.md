## Rules for editors

- **Do NOT replace any of the files with Google-Image-Search results** — those
  are almost always © protected and would break this clean setup.
- When adding new photos: pick from Wikimedia Commons, Unsplash, Pexels, or
  Pixabay — and update this file **in the same commit**.
- When a photo is removed from the Spots2 component, remove the corresponding
  file from this directory AND delete the row from the table above.
